# 2400033323-SkillEndSemExam-FEDF-

End sem lab exam project using React + Vite.

## Development

Install dependencies and run the dev server:

```powershell
npm install
npm run dev
```

Open the printed local URL (usually `http://localhost:5173`).

This setup includes HMR, ESLint, and a simple student dashboard layout (sidebar, topbar, and cards).
